# 帮助读者通过 Google 搜索中的首选来源找到您的网站

如果您是网站所有者，可以帮助受众群体在 Google 搜索中将您的发布内容视为首选来源。当用户将您的网站选为首选来源后，您的内容更有可能在焦点新闻中显示，以响应用户提出的相关新闻查询。

## 功能可用性

首选来源功能在全球范围内提供，适用于会触发“焦点新闻”功能的英文查询。

只有网域级和子网域级网站才有资格显示在来源偏好设置工具中。例如，https://www.example.com/ 和 https://code.example.com/ 符合首选来源条件，但子目录 https://www.example.com/blog 不符合。

```
https://www.example.com/
```

```
https://code.example.com/
```

```
https://www.example.com/blog
```

## 如何帮助用户将您的网站视为首选来源

如果您的网站显示在来源偏好设置工具中（您可以在该工具的搜索框中输入您的网站进行查看），您可以使用以下方法引导读者将您的网站选为首选来源：

- 在社交媒体帖子或宣传内容中添加深层链接。使用以下网址格式，让用户直接进入来源偏好设置工具中您的网站：
    https://google.com/preferences/source?q=Your_Website's_URL

      例如，如果您的网站为 https://example.com，请使用以下网址：

https://google.com/preferences/source?q=example.com
- 在您的网站上添加一个按钮，并与其他社交媒体号召性用语 (CTA) 一起显示。
    您可以下载并使用 Google 提供的这些按钮素材资源，也可以使用自己的按钮。

```
https://google.com/preferences/source?q=Your_Website's_URL
```

例如，如果您的网站为 https://example.com，请使用以下网址：

```
https://example.com
```

```
https://google.com/preferences/source?q=example.com
```
