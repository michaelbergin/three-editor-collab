## 

        Report issues with your site in Search Console

              If you're having trouble fixing a specific issue on your site in Search Console, you can use one of the following forms to report it directly to Google. You must be logged in to your Search Console Account to submit a report.

### 

  [

        Site verification issue

  ](https://support.google.com/webmasters/contact/sc_verification_issue_form)

            If you're having trouble verifying ownership of a domain property in Search Console, you can let us know.

### 

  [

        Security problem

  ](https://support.google.com/webmasters/contact/report_security_issues)

            If you're having trouble with persistant and unfixable security issues on your site in Search Console, you can let us know.

### 

  [

        Indexing bug

  ](https://support.google.com/webmasters/contact/indexing_issue_form)

            If you're having persistent issues getting your site indexed in Google Search and suspect a bug, you can report it to us.
