# 调试 Google 搜索流量下降问题

导致自然搜索流量下降的原因有很多，而您可能很难确定网站到底发生了什么。本指南介绍了如何使用 Search Console 效果报告和 Google 趋势来调查搜索流量下降的原因，以及如何解决该问题。

## 导致自然搜索流量下降的主要原因

如需了解哪些因素影响您的搜索流量，请查看以下图片中的概略图。它们大致显示了哪些因素可能影响您的流量，以及流量下降曲线图的形状。

算法更新、网站级安全问题或网络垃圾问题导致搜索流量大幅下降

季节性变化

以下各部分介绍了在分析流量下降问题时应调查的主要原因。此外，请务必查看 Search Console 数据异常页面，看看您的网站是否存在其中的任何情况。流量下降可能与数据处理方式变化或记录错误有关。

### 算法更新

Google 一直在改进其评估内容的方式，并相应地更新其搜索排名算法和内容呈现算法；核心更新和其他较小的更新可能会改变某些网页在 Google 搜索结果中的表现。 我们会在“排名更新列表”页面上发布关于系统的重要改进；请查看此页面，查看是否有适用于您网站的改进。

如果您怀疑流量下降是算法更新导致的，请务必明白您的内容可能不存在任何根本问题。如需确定是否需要进行更改，请在 Search Console 中查看您的热门网页并评估这些网页的排名情况：

- 排名出现小幅下降？例如，从第 2 位下降到第 4 位。
- 排名出现大幅下降？例如，从第 4 位下降到第 29 位。

### 排名出现小幅下降

排名出现小幅下降是指排名靠前的搜索结果排名出现小幅变化（例如，某个搜索查询的排名从第 2 位降至第 4 位）。在 Search Console 中，您可能会看到流量明显下降，但展示次数却没有发生太大变化。

排名随时可能会出现小幅波动（包括排名上移，而您无需执行任何操作）。事实上，如果您的网页本来已经表现良好，我们建议您不要进行重大更改。

### 排名出现大幅下降

排名出现大幅下降是指各种搜索字词的排名靠前搜索结果均明显排名下降的情况（例如，从前 10 位降至第 29 位）。

在这种情况下，请对整个网站（而不仅仅是单个网页）进行自我评估，确保它实用、可靠且以用户为本。
  如果您对网站进行了更改，可能需要一段时间才能看到效果：有些更改可能在几天内生效，而有些更改可能需要几个月才能生效。例如，我们的系统可能需要数月的时间才能确定某个网站正在长期生成有用的内容。一般来说，您可能需要等待几周的时间，在 Search Console 中再次分析您的网站，看看您所做的调整是否对排名产生了积极影响。

### 技术问题

技术问题即：可能阻止 Google 抓取网页、将网页编入索引或向用户提供网页的错误。例如，服务器可用性错误、robots.txt 抓取错误、“找不到网页”等。

请注意，此类问题可能是网站级别的（例如网站停止运行），或网页级别的（例如 noindex 标记放错了位置，该标记取决于 Google 能否抓取网页，这意味着流量下降速度可能较慢）。

```
noindex
```

查看“抓取统计信息”报告和“网页索引编制”报告，以确定是否检测到问题出现相应激增，这可能有助于您查明问题。

### 安全问题

如果您的网站受到安全威胁（例如恶意软件或钓鱼式攻击）的影响，Google 可能会在用户访问您的网站前向其显示警告或插页，这可能会导致搜索流量下降。

查看“安全问题”报告，了解 Google 是否在您的网站上检测到安全威胁。

### 网络垃圾问题

Google 会通过自动化系统以及在需要时借助人工审核检测违反 Google 搜索网络垃圾政策的行为，并可能采取人工处置措施。
  如果您的网站不符合Google 网页搜索的网络垃圾政策，您的内容可能会在搜索结果中排名较低，或者完全不显示在搜索结果中。

如果您怀疑网页因违反网络垃圾政策而出现流量下降，请参阅我们的网络垃圾政策，确保您没有采用自动化系统会检测到的网络垃圾行为。此外，请查看 Search Console 上的“人工处置措施”报告，确定您的网站是否已收到人工处置措施。

### 季节性趋势和兴趣变化

有时候，用户行为变化会改变对某些查询的需求，这可能是因新趋势或全年季节性变化所致。这意味着，您的流量可能因为外部因素影响而下降。

请使用效果报告找出点击次数和展示次数下降的查询，方法是应用过滤条件，一次仅包含一个查询（选择获得流量最高的查询）；然后，前往 Google 趋势查看这些查询，看看流量下降情况是只在您的网站上出现，还是在整个网络上都出现了。

### 网站迁移

如果您更改了网站上现有网页的网址，在 Google 重新抓取您的网站并将其重新编入索引时，您的网站的排名可能会出现波动。一般而言，对于中型网站，Google 可能需要几周的时间才会注意到这一变化；对于大型网站，则可能需要更长时间。

如果您在迁移网站后发现流量下降且没有恢复，请参阅“排查网站迁移问题”部分，了解迁移网站并更改网址时会出现的一些常见错误。

## 分析搜索流量下降规律

了解流量变化情况的最佳方法是查看 Search Console 效果报告中的主图表，其中汇总了大量信息。

如果展示次数和点击次数均出现下降，请查看最常见的原因列表，了解可能导致这种情况的最常见原因。如果展示次数没变，但点击次数下降了，可能是因为您没有生成最合适的网页标题和摘要，导致用户不了解您的网页中包含的内容，也可能是因为其他网站的富媒体搜索结果更具吸引力。

请查看搜索效果报告，并尝试对您的数据应用过滤条件，如以下几个部分所述。

### 将日期范围更改为包含 16 个月

选择图表顶部的日期过滤条件，然后选择过去 16 个月。
  这有助于您结合相关背景来分析流量下降情况，并确认流量下降不是因某个节日或趋势而每年都会发生。如果您想进一步扩大日期范围，可以使用 Search Analytics API 或批量数据导出来提取数据，并将其存储在您的系统中。

下面的图表显示了包含年度季节性变化（16 个月的数据）的效果图表。请注意，最近的下降情况与去年完全相同。

### 将流量下降时期与类似时期进行比较

选择图表顶部的日期过滤条件，选择比较标签页，然后选择对比过去 3 个月与前一对应时间段或对比过去 3 个月与去年同期。这有助于您发现到底是哪些方面发生了变化。请务必点击所有标签页，了解是否只有特定查询、网址、国家/地区、设备或搜索结果呈现的流量发生了变化（了解如何创建比较过滤条件）。

以下图表显示了三个月的效果对比情况。请注意，将实线（过去三个月）与虚线（再往前的三个月）进行比较时，流量下降非常明显。

### 单独分析各种搜索类型

选择图表顶部的搜索类型过滤条件，然后尝试使用其他可用选项。这有助于您了解所发现的流量下降情况具体发生在 Google 网页搜索、Google 图片、“视频”标签页还是“新闻”标签页。

### 监控您的网站在搜索结果中的平均排名

点击图表上方的平均排名。一般情况下，您不应过度关注绝对排名。展示次数和点击次数是衡量网站成功与否的最终指标。但是，如果您确实发现排名一直在明显下降，请尝试自行评估您的内容是否实用且可靠。

### 在受影响的网页中寻找规律

查看图表下方的网页表格，找出或许可以解释数据下降根源的规律。例如，一个重要的因素是了解下降情况是发生在整个网站上、一组网页上，还是只发生在网站中一个非常重要的网页上。为此，您可以将下降期与类似时期进行比较，并比较点击次数大幅下降的网页。选择点击次数差值，按流量下降幅度从高到低对网页进行排序。

如果是网站级问题，请查看“网页索引编制”报告。
  如果下降情况仅影响一组网页，请使用网址检查工具调查几个网页。

## 调查所在行业的总体趋势

如果您想进一步分析，可以通过 Google 趋势来了解流量下降是所在行业的总体趋势，还是自己网站的个别情况。流量下降可能是由两个主要因素导致的：

- 兴趣变化。如果发生导致用户搜索方式和搜索内容产生变化的重大事件，用户可能开始搜索不同的关键字，或将设备用于其他用途。此外，如果您在线销售某个品牌的商品，则相同的搜索查询可能会对应新的商品。
- 季节性变化。
    例如，美食网站的流量变化规律表明，与食品相关的查询具有很明显的季节性：用户经常在 1 月搜索节食菜谱、在 11 月搜索火鸡，在 12 月搜索香槟。不同的行业存在不同程度的季节性变化。

若要分析不同行业的趋势，您可以使用 Google 趋势获得大量未经过滤的真实 Google 搜索请求示例。这些请求经过了匿名化、分类和汇总。通过这种方式，Google 可以显示全球或细分到城市级别的不同主题的热度。

检查给您的网站带来流量的查询，确认这些查询是否在一年中的某些时期会明显减少。在本示例中，您可以看到三种趋势（查看数据）：

- “Turkey”一词的热度具有明显的季节性变化，每年 11 月都会达到峰值。
- “Chicken”一词的热度有一定的季节性变化，但幅度不大。
- “Coffee”一词的热度明显更加稳定；看来人们全年都需要喝咖啡。

此外，您不妨查看一些其他有趣的数据洞见，这可能有助于您提升搜索流量：

- 查看您所在地区的热门查询，并将这些查询对比为您带来流量的查询（参见 Search Console 效果报告）。如果您的流量中缺少某些查询，请检查您的网站是否提供相应主题的内容，并确保 Google 能顺利抓取此类内容并将其编入索引。
- 查看与重要主题相关的查询。这么做可能有助于您发现呈上升趋势的相关查询，从而方便您使网站做好准备，例如添加与这些新主题相关的内容。
