# 06-specialty - 特定网站指南

本目录用于存放针对特定类型网站的SEO指南文档。

## 分类说明

此分类涵盖以下特定网站类型的SEO最佳实践：

- **电商网站 (E-commerce)** - 在线商店、产品页面、购物车等
- **国际化/多语言网站 (International/Multilingual)** - hreflang、地理定位、多语言内容
- **新闻网站 (News)** - 新闻文章、焦点新闻、出版日期
- **视频网站 (Video)** - 视频内容、视频站点地图、视频结构化数据
- **特定行业网站 (Specialty)** - 特定垂直行业的SEO指南
